Grid System di Bootstrap

Bootstrap mette a disposizione una griglia per l'impaginazione responsive di contenuti.

Questa griglia può venire suddivisa fino a 12 colonne; ogni colonna, volendo, può anche venire suddivisa in ulteriori 12 slot e così via.

Le colonne stanno SEMPRE dentro le righe. Una row può stare o dentro un container, oppure dentro una col (al fine di creare ulteriori 12 colonne ad es.)
